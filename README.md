# KROZAUTOFISHER  
Minecraft autofishing bot — fast, configurable, lightweight.

---

## About the project
KrozAutoFisher is an automated Minecraft fishing bot that detects bobber movement based on pixel color variation.  
It reels fish automatically, supports adjustable sensitivity, autosell, hotkeys, and a live log UI.

No mods, no injection — works on any Minecraft version using pure screen detection.

## Usage

1. Open KrozAutoFisher.exe
2. Select your Minecraft window
3. Aim the crosshair at the fishing bobber
4. Press Start to begin
5. Press F8 (or assigned hotkey) to toggle on/off
6. Press Stop to end execution manually

---

## Recommended settings

| Setting | Value Range | Notes |
|---|---|---|
| Bite Sensitivity | 10–18 | Lower values react faster |
| Minimum Color Change | 30–40 | Detects bites reliably |
| AutoStop | 0–3 | Set to 0 to disable |
| Hotkey | F8 default | Fully configurable |
